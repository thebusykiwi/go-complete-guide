# Go Bank

A simple command-line banking application written in Go.  
This program allows users to check their balance, deposit money, withdraw money, and exit the application.

## Features

- **Check Balance:** View your current account balance.
- **Deposit Amount:** Add money to your account.
- **Withdraw Amount:** Withdraw money from your account (ensuring sufficient funds).
- **Exit:** Exit the application.
